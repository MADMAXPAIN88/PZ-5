# Какие преимущества дает использование Google Colab по сравнению с локальной средой разработки?
* Возможность использовать GPU (графический процессор) или TPU (процессор для машинного обучения Tensor Processing Unit) для выполнения вычислительно интенсивных задач, таких как обучение глубоких нейронных сетей.
# Какие типы ячеек существуют в Colab и для чего они используются?
* Ячейка кода (Code cell) - выполняется как Python код. Текстовая ячейка (Markdown cell) - для оформления текста, формул, описаний
# Как создать и вызвать пользовательскую функцию в Python?
* Пользовательская функция в Python создается с помощью ключевого слова def, вызывается функция написанием ее имени с круглыми скобками: имя_функции()
# Какие основные типы данных вы использовали в работе?
* int и str
